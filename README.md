# 🌸 LunaSkin — Cycle-Aware Skincare Platform

LunaSkin is a full-stack skincare web application that delivers **personalized skincare routines and product recommendations** based on menstrual cycle phases and individual skin types.  
The platform combines wellness intelligence with a modern and elegant user interface to empower users with data-driven beauty insights.

Built using **React, Next.js, and MongoDB**, LunaSkin focuses on personalization, automation, and user-centered design.

### 2. Install Dependencies
npm install
